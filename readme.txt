Коротокое описание решения для коллеги не-программиста или
менеджера:
1. Для добавления предмета нужно добаить запись в таблицу prizes данные о товаре
	label - название товара
	about - описание, его будет видеть пользователь
	image - путь к файлу относительно дериктории сайта
	rarity - пока что не используемая переменная, трогать не нужно
	count - количество данного товара
...обязательно 1 записью в данной таблице, с id=0 должны быть деньги

2. Для обработки призов нужно зайти в таблицу winners и обработать заказы с status - не отправлено, помните, что колонка
 статус будет видна пользователям.

Для программиста:
В папке есть дамп базы данных
Не добавлял пополнение с помощью онлайн кассы, потому что та привязывется к домену, но реализация, конечно, возможна
